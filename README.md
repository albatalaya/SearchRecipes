# Search Recipes

Implementation of a Recipes API for the Frontend project.

## Built with
- React
- [Edamam API](https://www.edamam.com/)

## Getting started
* Clone this repository: `git clone https://github.com/albatalaya/SearchRecipes.git`
* `cd SearchRecipes`
* `npm install` 
* `npm start`
